# NodeJS进阶

## 前言

本教程所有文章都基于node `v16.14.0`版本，内容包括：

- nodejs源码分析
- nodejs API使用要点
- 常见库、框架的实践

nodejs的源码分析，以源码为主，删去部分不重要的代码，辅以少量的的注释说明，做到调用顺序、代码结构清晰，抓住主干，容易理解。
